drill介绍

  drill 是第一个也是唯一一个不需要schema的SQL查询引擎。你可以直接使用dirll查询复杂、半结构化数据，而不需要提前进行数据清洗和转换。

安装

准备工作

  • JDK1.8以上
  • zookeeper
  • hadoop集群

    配置集群

  • 拷贝core-site.xml 到 drill的 conf目录下
  • 拷贝hdfs-site.xml 到 drill的 conf目录下

    启动drill

  • ./drill-embedded 单机模式
  • ./drillbit.sh (start|stop|graceful_stop|status|restart|autorestart) 集群模式

访问drill shell

  • bin/drill-conf 使用drill-override.conf指定的zk连接
  • 使用自定义方式连接drill(不改变配置文件的情况下使用特定插件连接)

退出drill shell

访问地址

http://localhost:8047
